# InvestWise - Stock & Finance Education Site

A professional, SEO-optimized static website providing free stock market and investment education content for global users.

## 🌐 Live Site

Deployed on Vercel: [https://investwise.vercel.app](https://investwise.vercel.app)

## 🎯 Purpose

InvestWise aims to provide high-quality, free financial education content. The site targets high AdSense CPC keywords in the finance niche ($2-5 per click).

## 📚 Content

12 comprehensive articles covering:
- Stock Market Basics
- Technical Analysis (Candlestick Charts, RSI, MACD)
- Fundamental Analysis (Financial Statements, EPS, ROE)
- Dividend Investing
- Risk Management
- ETF Investing
- Stock Screening
- Market Orders
- Portfolio Diversification
- Value Investing (Warren Buffett Method)
- Growth Stocks

## 🛠️ Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Custom styles with CSS variables
- **Tailwind CSS** - Via CDN for utility classes
- **JavaScript** - Vanilla JS for interactivity
- **Vercel** - Hosting & deployment
